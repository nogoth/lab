# lab

A scratch space for language experiments, algorithm work, and whatever I'm poking at this week.

## What's here

**`one-offs/`** — LeetCode problems worked through in Python: 2sum, 3sum, 4sum, sliding window, longest palindrome, quicksort, Fisher-Yates, a load balancer, problem 42 (trapping rain water), 238, 2747. Commit history shows the iteration; some problems have two or three attempts before they clicked.

**`leetcodes/`** — overflow from one-offs. Same idea.

**Root-level experiments** — various Python explorations:
- `mcp_client.py` / `mcp_server.py` — MCP protocol experimentation
- `lg.py` / `lag.py` — LangGraph/LangChain work
- `coqui-test.py` / `prompty.py` — voice/TTS experiments (Coqui, PyQT)
- `simple_server.py`, `main.py`, `test.py` — one-off scaffolding

## Philosophy

Nothing here is meant to be production code. It's a place to think out loud in code: algorithm intuition, new library exploration, dead ends included.
